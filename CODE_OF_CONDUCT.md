# قواعد السلوك - MedicalLabAnalyzer

## 🎯 رؤيتنا

نحن ملتزمون بتوفير بيئة مفتوحة ومضيافة للجميع، بغض النظر عن العمر، أو حجم الجسم، أو الإعاقة، أو العرق، أو الهوية الجنسية، أو التعبير الجنسي، أو مستوى الخبرة، أو التعليم، أو الحالة الاجتماعية والاقتصادية، أو الجنسية، أو المظهر الشخصي، أو العرق، أو الدين، أو التوجه الجنسي.

## 🌟 معاييرنا

### السلوك المتوقع
- استخدام لغة ترحيبية وشاملة
- احترام وجهات النظر والتجارب المختلفة
- قبول النقد البناء بأدب
- التركيز على ما هو أفضل للمجتمع
- إظهار التعاطف تجاه أعضاء المجتمع الآخرين

### السلوك غير المقبول
- استخدام لغة أو صور جنسية، أو اهتمام جنسي غير مرغوب فيه
- التصيد، أو التعليقات المهينة/التحقيرية، أو الهجمات الشخصية أو السياسية
- المضايقة العامة أو الخاصة
- نشر معلومات خاصة للآخرين، مثل العناوين الفعلية أو الإلكترونية، دون إذن صريح
- سلوك آخر قد يعتبر غير لائق في بيئة مهنية

## 🛠️ مسؤولياتنا

### مسؤوليات المطورين
- الحفاظ على بيئة ترحيبية ومهنية
- معالجة السلوك غير المقبول بسرعة وعادلة
- التواصل مع قادة المشروع عند الحاجة

### مسؤوليات قادة المشروع
- توضيح معايير السلوك المقبول
- اتخاذ إجراءات تصحيحية مناسبة وعادلة استجابة لأي حالات سلوك غير مقبول
- إزالة أو تحرير أو رفض التعليقات والالتزامات والكود والتحريرات الأخرى التي لا تتوافق مع قواعد السلوك هذه

## 🚨 الإبلاغ عن الحوادث

### كيفية الإبلاغ
إذا واجهت سلوكاً مسيئاً أو مضايقاً، أو لديك مخاوف أخرى، يرجى الاتصال بفريق المشروع عبر:

- **البريد الإلكتروني**: conduct@medicallabanalyzer.com
- **GitHub Issues**: إنشاء issue خاص (سيتم إغلاقه للجمهور)

### معلومات مطلوبة
يرجى تضمين:
- وصف للحادث
- تاريخ ووقت الحادث
- أسماء الأشخاص المشاركين
- أي أدلة متاحة (شاشات، رسائل، إلخ)

## ⚖️ الإجراءات التصحيحية

### الإجراءات الممكنة
- تحذير خاص أو عام
- حظر مؤقت أو دائم من المشاركة
- إزالة التعليقات أو المحتوى المسيء
- إزالة المساهمات من سجل المشروع

### معايير القرار
- شدة الحادث
- تكرار السلوك
- نية الفاعل
- تأثير السلوك على المجتمع

## 🔄 الاستئناف

### حق الاستئناف
إذا كنت تعتقد أنك تعرضت لعقوبة غير عادلة، يمكنك الاستئناف عبر:

- **البريد الإلكتروني**: appeals@medicallabanalyzer.com
- **الموضوع**: [APPEAL] وصف مختصر للقضية

### عملية الاستئناف
- مراجعة من قبل لجنة مستقلة
- قرار نهائي خلال 14 يوماً
- إشعار مكتوب بالقرار
